Database access

There are two possibilities to access the database:

  • Via the web using phppgadmin at https://eurodeer2.fmach.it/phppgadmin/
  • Using a PostgreSQL client (like DBeaver, pgAdmin, R, Python); the parameters to use are:
    • Host: eurodeer2.fmach.it
    • Port: 5432
    • Database: DATABASE_YOU_WANT_USE
    • User: YOUR_USER
    • Password: YOUR_PASSWORD
